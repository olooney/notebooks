# JupyterLite Notebooks
### Oran Looney

This repo uses [JupyterLite][JL] to publish some of my open source Jupyter
notebooks to GitHub Pages:

* https://olooney.github.io/notebooks 

It was cloned from their [demo repo][DR] following their
[recommended deployment guide.][RDG]

JupyterLite runs everything entirely inside your own browser; nothing is
sent to my servers. You can view and interact with the notebooks on the
site, or download them and run them yourself. All of the notebooks are
made available with the permissive [BSD-3 license][BSD].


[JL]: https://jupyterlite.readthedocs.io/en/stable/
[DR]: https://github.com/jupyterlite/demo
[RDG]: https://jupyterlite.readthedocs.io/en/latest/quickstart/deploy.html
[BSD]: ./LICENSE

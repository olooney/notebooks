# JupyterLite Notebooks
### by Oran Looney

This repo uses [JupyterLite][JL] to publish some open source Jupyter notebooks
to GitHub Pages:

* https://notebooks.oranlooney.com/

A good place to start is [classification_anomaly.ipynb][CA], which has a
particularly interesting interactive visualization of a machine learning
algorithm with anomaly rejection built with IPyWidgets.

JupyterLite runs everything entirely inside your own browser; nothing is
sent to my servers. You can view and interact with the notebooks on the
site, or download them and run them yourself. All of the notebooks are
made available with the permissive [BSD-3 license][BSD].

This repo was cloned from their [demo repo][DR] following their
[recommended deployment guide.][RDG]
